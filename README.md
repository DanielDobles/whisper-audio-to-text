# Whisper Audio to Text (Kaggle Edition)

Este proyecto contiene un cuaderno de Jupyter diseñado para ejecutarse en Kaggle, utilizando el modelo **Whisper** de OpenAI para la transcripción de audio a texto aprovechando la aceleración por GPU.

## Características
- Instalación de dependencias en el entorno de Kaggle.
- Carga de modelos Whisper (tiny, base, small, medium, large).
- Transcripción de archivos de audio locales.
- Soporte para múltiples idiomas.

## Uso
Sube el archivo `.ipynb` a un nuevo Dataset o Notebook en Kaggle, asegúrate de activar la **GPU T4 x2** o **P100** en la configuración de la sesión para un rendimiento óptimo.
